## 📌 About the Project

This portfolio is designed to highlight my journey as a developer, including my technical skills, real-world projects, and achievements. It serves as a central place for recruiters, collaborators, and clients to learn more about my work.

The design focuses on **clean UI, smooth user experience, and responsive layout across all devices**.

---

## ⚙️ Features

- 🏠 Modern landing page with personal introduction  
- 👨‍💻 About section highlighting skills and background  
- 🧠 Skills showcase (Frontend, Backend, Tools)  
- 📂 Projects section with real-world applications  
- 📄 Resume download option  
- 📬 Contact form integration  
- 🌙 Responsive design for all screen sizes  
- ⚡ Smooth animations and transitions
